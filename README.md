# בוחן עצמי בנוירואנטומיה

אתר סטטי (HTML/CSS/JS) לתרגול מבחן בנוירואנטומיה — **רק מצב בוחן עצמי**.

## מה יש באתר

- **100 שאלות אמריקאיות** ב-10 נושאים, בהשראת מבחני העבר (2023–2025) וחומר הקורס
- בחירת נושא (או כל החומר יחד) וכמות שאלות לסבב
- משוב מיידי והסבר לכל תשובה
- סיכום בסוף + סקירת טעויות

## איך פותחים

```bash
cd neuroanatomy-quiz
python3 -m http.server 8000
# ואז http://localhost:8000
```

או פשוט לפתוח את `index.html` בדפדפן.

## מבנה

| קובץ | תוכן |
|---|---|
| `index.html` | מסך פתיחה, בוחן, תוצאות |
| `assets/questions.js` | מאגר 100 השאלות |
| `assets/app.js` | לוגיקת הבוחן |
| `assets/style.css` | עיצוב RTL |

## נושאים (10)

תאים והתפתחות · מוח גדול ושפה · לימבית · דיאנצפלון/אוטונומית · עצבים קרניאליים · גזע מוח · צרבלום · מוטורי · סומטוסנסורי · קרומים/CSF/דם
